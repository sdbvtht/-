心率实时监控工具

技术规格  
• 开发语言：Python 3.14  
• WebSocket 端口：6667  
• Web 服务端口：20888  
• 系统要求：Windows 10/11  

手机版APP来自BILIBILI-zj1d

功能特点：  
 • 实时接收并显示心率数据  
 • 本地网页同步展示，支持多设备访问  
 • 可自定义服务器 IP 地址  
 • 心率数值颜色随区间变化  
 • 自动重连，稳定可靠  
 • 可用于OBS直播
